# ASUS_K555LP_Hackintosh

华硕K555LP5200U 目前适配10.15.4

## 笔记本配置
| 硬件名称| 配置详情 |
| :----: | :---- |
| 主板 | Asus X555LJ |
| CPU | Intel core I5-5200U  2.2GHZ |
| 集成显卡 | HD 5500 |
| 独立显卡 | GeForce 920M |
| 有线网卡 | Realtek RTL8168/8111 PCI-E (Realtek RTL8111) |
| 无线网卡 | BCM43142 （已更换为BCM94352HMB）|
| 声卡 | Realtek ALC233 |

## 成功驱动
+ 核显HD5500 
+ 声卡
+ 有线网卡
+ 无线网卡（BCM43142基本无解 更换BCM94352HMB完美解决）
+ 蓝牙（BCM94352HMB）
+ 隔空投送（BCM94352HMB）
+ 定位（BCM94352HMB）
+ 亮度调节
+ 电量显示
+ 触摸板
+ USB定制
+ CPU睿频
